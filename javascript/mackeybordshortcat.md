# Mac Keyboard Shortcuts

This file contains a comprehensive list of common keyboard shortcuts for macOS and VS Code.

## General macOS Shortcuts

- **Cmd + C**: Copy
- **Cmd + V**: Paste
- **Cmd + X**: Cut
- **Cmd + Z**: Undo
- **Cmd + Shift + Z**: Redo
- **Cmd + A**: Select all
- **Cmd + F**: Find
- **Cmd + S**: Save
- **Cmd + W**: Close window
- **Cmd + Q**: Quit application
- **Cmd + Tab**: Switch between applications
- **Cmd + Space**: Open Spotlight search
- **Cmd + Shift + 3**: Screenshot entire screen
- **Cmd + Shift + 4**: Screenshot selected area
- **Cmd + Shift + 5**: Open screenshot tool
- **Cmd + Option + Esc**: Force quit applications
- **Cmd + H**: Hide window
- **Cmd + M**: Minimize window
- **Cmd + N**: New window/document
- **Cmd + O**: Open file
- **Cmd + P**: Print
- **Cmd + ,**: Open preferences
- **Fn + F11**: Show desktop
- **Ctrl + Cmd + Q**: Lock screen

## VS Code Shortcuts (on Mac)

- **Cmd + /**: Toggle line comment
- **Cmd + Shift + /**: Toggle block comment
- **Cmd + ]**: Indent line
- **Cmd + [**]: Outdent line
- **Cmd + Shift + P**: Open command palette
- **Cmd + P**: Quick open file
- **Cmd + Shift + N**: New window
- **Cmd + W**: Close window
- **Cmd + S**: Save file
- **Cmd + Shift + S**: Save as
- **Cmd + Z**: Undo
- **Cmd + Shift + Z**: Redo
- **Cmd + A**: Select all
- **Cmd + F**: Find
- **Cmd + Shift + F**: Find in files
- **Cmd + H**: Replace
- **Cmd + Shift + H**: Replace in files
- **Cmd + G**: Find next
- **Cmd + Shift + G**: Find previous
- **Cmd + D**: Add selection to next find match
- **Cmd + Shift + L**: Select all occurrences of current selection
- **Cmd + Shift + Enter**: Insert line above
- **Cmd + Enter**: Insert line below
- **Cmd + Shift + K**: Delete line
- **Cmd + Shift + \**: Jump to matching bracket
- **Cmd + Shift + [**]: Fold code block
- **Cmd + Shift + ]**: Unfold code block
- **Cmd + K Cmd + 0**: Fold all
- **Cmd + K Cmd + J**: Unfold all
- **Cmd + B**: Toggle sidebar
- **Cmd + Shift + E**: Focus explorer
- **Cmd + Shift + F**: Focus search
- **Cmd + Shift + G**: Focus source control
- **Cmd + Shift + D**: Focus debug
- **Cmd + Shift + X**: Focus extensions
- **Cmd + J**: Toggle panel
- **Cmd + Shift + M**: Toggle problems panel
- **Cmd + Shift + U**: Toggle output panel
- **Cmd + Shift + Y**: Toggle debug console
- **Cmd + `**: Switch between editor groups
- **Cmd + 1-9**: Switch to editor tab 1-9
- **Cmd + Shift + [**]: Previous editor
- **Cmd + Shift + ]**: Next editor
- **Cmd + K Cmd + S**: Open keyboard shortcuts
- **Cmd + ,**: Open settings
- **Cmd + Shift + P**: Open command palette (alternative)
- **F5**: Start debugging
- **F10**: Step over
- **F11**: Step into
- **Shift + F11**: Step out
- **Ctrl + Shift + F5**: Restart debugging

## Text Editing Shortcuts

- **Option + Left/Right Arrow**: Move cursor by word
- **Cmd + Left/Right Arrow**: Move to beginning/end of line
- **Cmd + Up/Down Arrow**: Move to beginning/end of file
- **Shift + Arrow Keys**: Select text
- **Option + Shift + Left/Right Arrow**: Select by word
- **Cmd + Shift + Left/Right Arrow**: Select to beginning/end of line
- **Cmd + Shift + Up/Down Arrow**: Select to beginning/end of file
- **Cmd + Delete**: Delete to beginning of line
- **Option + Delete**: Delete word backward
- **Fn + Delete**: Delete forward (equivalent to Del key)

## Browser Shortcuts (Safari/Chrome)

- **Cmd + T**: New tab
- **Cmd + W**: Close tab
- **Cmd + Shift + T**: Reopen closed tab
- **Cmd + R**: Refresh page
- **Cmd + L**: Focus address bar
- **Cmd + Shift + R**: Hard refresh
- **Cmd + [+/-]**: Zoom in/out
- **Cmd + 0**: Reset zoom

Note: Some shortcuts may vary slightly between applications. Check the app's menu for specific shortcuts.
